How Cloud Print Mesh™ works

A simple, globally distributed pipeline that connects your applications, AI-powered label templates, and on-prem printers with sub-second end-to-end latency.

End-to-end flow

From app to label in a single, resilient pipeline:

[ User / ERP / WMS ] │ (Describe or trigger label) ▼ [ AI Label Designer ] │ (Generate + validate ZPL) ▼ [ Cloudflare Worker ] │ (Route & log job) ▼ [ Durable Object Router ] │ (Find target LMG + printer) ▼ WebSocket [ LMG in Warehouse ] │ TCP 9100 ▼ [ Zebra Printer ] → Physical label

Key components

  • AI Label Generator – natural language to ZPL in seconds
  • Cloudflare Edge – job intake, validation, routing
  • Durable Objects – track LMG + printer state
  • LMG (Local Management Gateway) – secure link to LAN printers
  • Print Control Dashboard – global observability & replay
